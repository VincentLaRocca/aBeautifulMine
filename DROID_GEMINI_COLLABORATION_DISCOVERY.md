### 4. Format Issue Identified

**Gemini's Output Format:**
```json
{
  "bitcoinDominanceFAQ": [
    {
      "question": "...",
      "answer": "..."
    }
  ]
}
```

**Required Format:**
```json
{
  "indicator_name": "bitcoin_dominance",
  "display_name": "Bitcoin Dominance (BTC.D)",
  "session_number": 35,
  "category": "dominance_metrics",
  "total_pairs": 100,
  "qa_pairs": [
    {
      "pair_number": 1,
      "question": "...",
      "answer": "..."
    }
  ]
}
```

**Missing:**
- Metadata fields (indicator_name, display_name, session_number, category)
- pair_number in each Q&A
- Different array name ("bitcoinDominanceFAQ" vs "qa_pairs")

**Impact:** Minor - content is good, just needs structural conversion
