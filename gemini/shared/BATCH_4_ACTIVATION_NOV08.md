## Research Tips

### Parabolic SAR
- Focus on the acceleration factor (AF) and how it changes
- Crypto-specific: High volatility means SAR flips frequently
- Trading strategies: Exit on SAR flip, trailing stop method
- Common mistakes: Using default settings (0.02 AF) in crypto without adjustment

### Ichimoku Cloud System
**This is a complete trading system, not just individual indicators!**

**For Tenkan-sen (Conversion Line)**:
- Fast-moving line (9-period)
- Acts like a faster moving average
- Crossovers with Kijun-sen are key signals
- Above price = resistance, below = support

**For Kijun-sen (Base Line)**:
- Slower confirmation line (26-period)
- More significant support/resistance
- Price above = bullish, below = bearish
- Flat Kijun = ranging market

**For Senkou Span A (Leading Span A)**:
- Average of Tenkan + Kijun, plotted 26 periods ahead
- Faster-moving cloud boundary
- Creates the "cloud" with Senkou Span B
- Thicker cloud = stronger support/resistance

**For Senkou Span B (Leading Span B)**:
- 52-period midpoint, plotted 26 periods ahead
- Slower-moving cloud boundary
- Major support/resistance level
- When price is above cloud = strong uptrend

**Full System Trading**:
- Include in answers: How all components work together
- Entry: Price above cloud + Tenkan crosses above Kijun
- Exit: Price enters cloud or Tenkan crosses below Kijun
- Strength: Thick cloud = strong trend

### Keltner Channels
- ATR-based (unlike Bollinger Bands which use standard deviation)
- Less responsive to volatility spikes
- Breakout strategies: Price above upper channel = bullish breakout
- Comparison to Bollinger Bands (mention differences)
